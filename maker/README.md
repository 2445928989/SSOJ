# 测试数据生成工具 (Maker)

Maker 是一个用于快速生成题目测试数据的工具集。通过编写简单的生成逻辑和标准解法，即可自动产生成对的 `.in` 和 `.out` 文件。

## 核心组件

- `gen.cpp`: 数据生成器。在此编写随机数据生成逻辑，并将结果输出到标准输出。
- `sol.cpp`: 标准解法（标程）。在此编写题目的正确解法，从标准输入读取数据，输出到标准输出。
- `maker.sh`: 自动化脚本。负责编译上述程序，并循环调用生成器产生指定数量的测试数据。
- `.env`: 配置文件。用于设置生成数据的组数及存放目录。

## 使用流程

1. **编写生成逻辑**：在 `gen.cpp` 中实现单组数据的随机化生成。
2. **编写标准解法**：在 `sol.cpp` 中实现题目的 AC 代码。
3. **配置参数**：修改 `.env` 文件中的 `COUNT` 参数（默认为 5）。
4. **运行脚本**：
   ```bash
   chmod +x maker.sh
   ./maker.sh
   ```
5. **获取数据**：生成的测试点将保存在指定的输入和输出目录下（默认为 `./in` 和 `./out`）。

## 注意事项

- 随机种子：`gen.cpp` 建议使用 `std::random_device` 或基于时间的种子，以确保每组数据不同。
- 脚本兼容性：`maker.sh` 适用于 Linux/macOS 环境。Windows 用户可参考 `maker.bat`。
- 批量上传：生成的数据可以直接打包成 ZIP 格式，通过 SSOJ 管理后台上传。
